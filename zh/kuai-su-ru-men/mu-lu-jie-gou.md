# 项目目录结构

项目目录结构如下：

```
├── LICENSE
├── README.md
├── _config.yml
├── bin
│   └── console.php
├── composer.json
├── composer.lock
├── config
│   ├── app.yml
│   ├── config.yml
│   ├── example.app.yml
│   ├── example.dbs.yml
│   ├── example.migrations-db.php
│   ├── example.parameters.yml
│   ├── migrations.yml
│   ├── parameters.yml
│   ├── routes.yml
│   └── routes_test.yml
├── docs
├── migrations
├── phpunit.xml.dist
├── src
│   ├── App
│   ├── Common
│   ├── FrontBundle
│   └── Service
├── tests
│   ├── AppTest.php
│   ├── bootstrap.php
│   ├── config
│   ├── src
│   └── web
├── var
│   ├── cache
│   └── logs
├── vendor
│   └── ...
└── web
    ├── assets
    ├── bootstrap.php
    ├── favicon.ico
    ├── index.php
    └── views
```
